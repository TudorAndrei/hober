# Hober

![](misc/logo.png)

## Documents

The documents must be placed in the `./documents` folder.

## Query

The queries must be placed in the `query/queries.txt` file. Each search on a new line

## Build project

```
./graldew build
```

## To index files

```
./gradlew index
```

## To search

```
./gradlew search 
```
